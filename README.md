# androidTube
Projeto desenvolvido no Android Studio com foco em reprodução de vídeos em uma interface simples e interativa.
A aplicação possui uma tela principal onde o usuário pode escolher entre três vídeos diferentes através de botões dedicados.
O player permite controlar a reprodução com funções de play e pause em tempo real.
Também conta com uma seekbar para avançar ou retroceder o vídeo de forma dinâmica.
O tempo de reprodução é atualizado automaticamente durante a execução do vídeo.
O objetivo do projeto é demonstrar o uso de componentes multimídia e manipulação de vídeo no Android.
